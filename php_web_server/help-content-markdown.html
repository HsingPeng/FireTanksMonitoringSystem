消防水箱远程监控系统

FireTanksMonitoringSystem

通过对课题的研究与实践，深入研究了解，开发出一套城市消防远程监控系统的监控端配合监测终端可以提高消防部队快速反应，提高社会单位自身消防安全管理水平和全社会防控火灾能力。

本系统诣在实时监控水箱以下三项参数：

  • 水压
  • 水温
  • 水位

安全范围

此安全范围通过互联网收集整理，仅供参考

种类 安全 警告 危险
水压 2->正常 3->充足 1->较低
水温 6~60℃ 4~6℃ <3℃
水位 2->较低 3->正常 4->充足 1->过低

服务器相关

  • WEB前端：Bootstrap框架
  • 服务器后端：PHP
  • 数据库：MySQL
